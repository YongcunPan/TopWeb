售前服务介绍

所在位置：首页>解决方案>行业医院

NO.1 售前服务内容

我们配备了专业技术人员为您解答在插座和转换器以及机房专用PDU产品定制使用和安装情况及产品实现的可行性，为您解决采购电源带来的烦恼，信息化的今天，信息传递在我们生活中每天每小时每分钟都在你我他之间传递，要保证信息传递的及时和准确，所以要确保机房电源不能间断，通过质量可靠的UPS给服务器供电，当然也需要高质量的PDU来为设备进行电源分配，如果您现在还在使用可靠性差的普通民用插座，如果您需要对机房进行改造或增加新的设备，我们会有专业技术人员为您量身定制。

NO.2 我们可以为您提供以下服务

  • 1·常规订制品和PDU订制品技术咨询，疑难问题的及时解答。
  • 2·机房现场勘测，量身订制，解决PDU产品配置和安装问题。
  • 3·依据您的订制需求为您绘制图纸（平面图纸和效果图纸）。
  • 4·为您提供产品的技术参数。

NO.3 定制流程

  • 分析客户环境：客户行业特征 / 客户机房环境 / 机柜内设备
  • 产品功能需求：电涌防护 / 空气开关 / 漏电保护 / 数字电流显示 / 其它功能
  • 实用性需求：长度 / 孔型模式 / 插头模式 / 进线设计 / 安装需求
  • 购买需求：价格因素 / 时间因素 / 送货及服务
